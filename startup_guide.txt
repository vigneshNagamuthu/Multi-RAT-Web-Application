**IMPORTANT**
If your system requires a sudo password, please edit the visudo file by:
1) opening terminal and enter
    sudo visudo
2) add the following lines at the end of the file:
    @includedir /etc/sudoers.d
    <username>    ALL=(ALL) NOPASSWD: /usr/sbin/ip link set *
    <username>    ALL=(ALL) NOPASSWD: /usr/sbin/sysctl *

\\How to start web app locally//
Prerequisites:
 - Java 21 JDK installed (verify with: java -version and javac -version, both should show 21.x)
 - Maven installed (verify with: mvn -v)
 - Node.js + npm installed (verify with: node -v and npm -v)
   If missing, install via nvm: curl -fsSL https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
   Then: export NVM_DIR="$HOME/.nvm" && [ -s "$NVM_DIR/nvm.sh" ] && . "$NVM_DIR/nvm.sh" && nvm install --lts

Backend steps:
1) open terminal and go to /backend directory
2) (optional first time) mvn -q -e dependency:go-offline
3) build the project:
    mvn clean install
4) run the backend:
    mvn spring-boot:run
   (Alternatively run the jar after build: java -jar target/backend-0.0.1-SNAPSHOT.jar)
5) backend is running on default port (check console for 'Started BackendApplication')

Frontend steps:
6) open a new terminal and go to /frontend directory
7) install dependencies (first time only):
    npm install
   (react-router-dom already in dependencies if package.json lists it; otherwise install explicitly.)
8) start the frontend dev server:
    npm start
9) open browser at: http://localhost:3000/

Troubleshooting:
 - If you see 'release version XX not supported', ensure java -version is 21+.
 - If port conflicts occur, export SERVER_PORT=8081 before running spring-boot:run.
 - Clean build issues: mvn clean; delete ~/.m2/repository if corruption suspected.
 - Video not showing: Check browser camera permission and WS console logs.
 - Sensor RTT stays null: Ensure /sensor WebSocket connected (network tab) and click Ping again.

New Prototype Pages:
 - /video : Sends webcam frames (JPEG over WebSocket) and displays rebroadcast stream.
 - /sensor : Sends mock sensor JSON messages and measures ping RTT.
